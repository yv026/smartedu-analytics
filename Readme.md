SmartEdu App

Description

SmartEdu est une application développée dans le cadre d’un projet en analyse de données. Elle permet d’exécuter différentes techniques de data science comme la régression linéaire simple et multiple, la classification supervisée et non supervisée, ainsi que l’analyse en composantes principales (ACP). L’objectif est d’aider les étudiants à mieux comprendre et appliquer les méthodes d’analyse de données de manière simple et interactive.

Fonctionnalités

L’application propose plusieurs fonctionnalités principales : la régression linéaire pour prédire des valeurs, la classification supervisée pour organiser des données étiquetées, la classification non supervisée pour regrouper des données similaires, et la réduction de dimensionnalité grâce à l’ACP. Elle permet également l’importation de jeux de données et la visualisation des résultats d’analyse.

Technologies utilisées

Le projet est développé en Python avec l’utilisation de bibliothèques comme NumPy, Pandas et Scikit-learn pour le traitement et l’analyse des données.

Installation

Cloner le projet avec git clone, installer les dépendances avec pip install -r requirements.txt, puis lancer l’application avec python app.py.

Utilisation

Importer un fichier de données, choisir une méthode d’analyse (régression, classification ou ACP), puis exécuter l’algorithme pour afficher les résultats.
